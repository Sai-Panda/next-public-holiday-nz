# Claude-specific instructions

- This project is built mobile-first, always. Design, implement, and verify every layout at mobile viewport widths first, then progressively enhance for tablet/desktop breakpoints. Never design desktop-first and retrofit mobile.
- When testing UI changes in a browser, resize/check the mobile viewport before (or in addition to) desktop.
- Follow the shared working agreement in `.github/copilot-instructions.md` and current focus in `.github/copilot-context.md` for everything else (tech stack, validation commands, code style).
